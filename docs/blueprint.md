# **App Name**: AssetLoaderAds

## Core Features:

- Index Loader: Load `index.html` from the assets folder on application start.
- Admob Banner: Display a banner ad from AdMob at the top or bottom of the screen.
- Admob Interstitial: Show an interstitial ad from AdMob at app intervals.

## Style Guidelines:

- Background color: Light gray (#f0f0f0) to ensure the HTML content from assets folder is clearly visible.
- Primary color: Blue (#3498db) for interactive elements like buttons.
- Accent color: Green (#2ecc71) for key actions and highlights.
- Font: 'Inter', a sans-serif font, to ensure legibility.
- Ensure Admob banner doesn't overlap important content loaded from `index.html`.
